# helix_splunk_lab
Развернута лаборатория SOC, оснащенная Splunk, для агрегации логов Windows и выявления попыток брутфорса

Лаборатория SOC: Splunk и логи Windows

Данный проект демонстрирует создание базовой лаборатории SOC с использованием Splunk и сбор логов (журналов событий) Windows для мониторинга безопасности

Используемые технологии:
Splunk Enterprise
Splunk Universal Forwarder
Windows 10 VM
VirtualBox

Настройка лаборатории:
Splunk Enterprise установлен на хост-машину
Universal Forwarder установлен на виртуальную машину Windows
Настроена пересылка журналов в Splunk (порт 9997)
Произведен сбор журналов событий Windows (журнал безопасности, журнал системы)

Пример обнаружения
Обнаружение брутфорса (неудачные попытки входа)
EventCode=4625
| stats count by host
| where count > 3

Результаты:
Успешно проиндексированы журналы Windows в Splunk
Обнаружены множественные неудачные попытки входа
Разработана первоначальная логика обнаружения
